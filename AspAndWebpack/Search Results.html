                                                                                                                                                  • new options to create and name your own custom data grids
                                                                                                                                                  • ability to export all subscribed markets in one spreadsheet

                                                                                                                                                  The new site also retains all the old features of Markets Forecast. We hope you will like the new site but we will be keeping the old Markets Forecast for a period while you get used to the new site.
